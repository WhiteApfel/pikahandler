Welcome to PikaHandler's documentation!
=======================================

Это удобный модуль для комфортной обработки сообщений из очереди RabbitMQ.
Модуль добавляет декораторы-хендлеры, которые улучшают визуальную структуру кода.
Звёздочка на гитхабе сделает меня, кстати, счастливым =-)
